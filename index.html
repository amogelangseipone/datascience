types of cloud services

Software as a service [] – the service provider offers turnkey applications that run in the cloud []. The client can run the applications internally [] and customise certain aspects of the applications but cannot make changes to the application itself []. Platform as a service [] – the service provider offers the capability to build and deploy consumer created applications using the provider’s cloud infrastructure []. The client can build, deploy and manage applications using the provider’s tools, languages and interfaces []. Infrastructure as a service [] – the service provider offers the ability to provision their own resources on demand []. The consumer can add or remove the resources as needed [].
